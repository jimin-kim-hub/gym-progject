# 🏋️ Gym-Now: 실시간 헬스장 혼잡도 알림 서비스

> **"지금 가면 기구 쓸 수 있을까?"** > 반복되는 혼잡도 문의를 챗봇으로 해결하고, 관리자와 회원 모두의 시간을 아끼는 솔루션입니다.

---

## 🚀 Key Features
- **지점별 관리자 대시보드**: 모바일에서 버튼 하나로 현재 인원 즉시 업데이트
- **카카오톡 챗봇 연동**: 사용자가 카톡 채널을 통해 실시간 인원 확인 (API 연동)
- **지점별 보안 시스템**: 각 헬스장 데이터 격리 및 관리자 인증 로직 구현
- **전체 로그 관리**: 지민(Master) 전용 비밀번호를 통한 통합 기록 모니터링

## 🛠 Tech Stack
- **Framework**: FastAPI (Python)
- **Database**: PostgreSQL / SQLAlchemy
- **Interface**: Kakao API / HTML5 / CSS3
- **Deployment**: Render

## 💡 PM 인사이트 (Market Validation)
- **가설**: 소규모 헬스장은 피크타임 혼잡도가 회원 만족도에 큰 영향을 미칠 것이다.
- **실행**: 대학가 인근 15개 지점을 대상으로 직접 B2B 영업 및 제안서 발송.
- **배운 점**: 제품의 가치보다 '관리자의 입력 수고'라는 운영 장벽이 높음을 파악. 이를 통해 'AI가 해결하지 못하는 사람의 판단이 필요한 영역'에 대한 차기 비즈니스 모델 아이디어를 확보함.
